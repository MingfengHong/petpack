快速开始 ​

本页从下载安装开始，带你完成一次真实宠物导入、检查和打包。一般不需要准备开发环境。

1. 下载适合当前设备的 Studio ​

前往 GitHub Releases：

设备下载文件
Windows 10/11 x64PetPack Studio_*_x64-setup.exe
Apple Silicon Mac文件名包含 aarch64 的 DMG
Intel Mac文件名包含 x64 或 x86_64 的 DMG
通用 Linux 桌面.AppImage
Debian / Ubuntu.deb

当前社区构建没有商业代码签名。若系统显示 SmartScreen、Gatekeeper 或执行权限提示，请先阅读常见问题与排错。

2. 准备宠物包 ​

最简单的结构如下：

text
my-pet/
├── pet.json
└── spritesheet.webp
1
2
3

也可以使用：

  • 包含上述文件的 ZIP；
  • pet.json 或 spritesheet 文件本身；
  • 只包含一个宠物子目录的父文件夹；
  • Petdex slug，例如 boba；
  • Petdex 页面地址，例如 https://petdex.dev/pets/boba。

图集尺寸和清单字段见宠物包格式。

3. 导入 ​

启动 Studio 后，在“导入宠物”区域：

  1. 点击“选择文件夹”或“选择 ZIP / 文件”；
  2. 也可以把文件夹或 ZIP 直接拖进窗口；
  3. 使用 Petdex 时，在输入框填写 slug 或页面地址，然后点击“识别并导入”。

导入完成后，Studio 会显示识别格式和图集尺寸。发现阻断问题时，发布按钮保持禁用。

4. 检查与预览 ​

在“检查与预览”区域确认：

  • 图集尺寸和行列数量正确；
  • 使用帧有内容，未使用帧透明；
  • 清单中的名称、版本和 spritesheet 路径正确；
  • 待机、移动、挥手、跳跃、失落、等待、工作和检查动作能够正常播放。

点击“桌面试玩”可以在真正的透明桌宠窗口中检查拖动、缩放和动画效果。

5. 选择发布方式 ​

校验通过后，“发布桌宠”区域会变为“可以发布”。

当前平台成品 ​

适合同平台直接分发。点击“选择目录并生成”后会得到：

  • 原生桌宠运行程序；
  • petpack.bundle/ 宠物资源；
  • 使用说明和第三方声明；
  • 可直接发送的 ZIP。

跨平台构建包 ​

用于把同一个宠物分发到其他操作系统。构建包包含宠物资源、一键入口和目标设备使用说明；缺少对应 builder 时会在首次运行自动下载。详细流程见跨平台构建与分发。

6. 运行桌宠 ​

解压生成的 ZIP，启动其中的桌宠程序：

  • 桌宠窗口透明、无边框并保持置顶；
  • 从底部把手拖动位置；
  • 光标进入宠物区域后显示缩放、动作、置顶和退出工具栏；
  • 托盘菜单可以显示/隐藏、切换动作和调整大小；
  • 桌宠不显示在任务栏，关闭或退出应使用悬浮工具栏或托盘菜单。
