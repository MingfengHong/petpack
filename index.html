PetPack Studio让你的 Codex 宠物走出 Codex

导入、校验并预览 Codex 或 Petdex 宠物，将它打包成不依赖 Codex 的 Windows、macOS 和 Linux 独立桌宠。

GitHub 仓库

导入与识别

支持 Codex v2、Codex/Petdex v1、本地文件夹、ZIP、pet.json、spritesheet 和 Petdex 页面链接。

校验与预览

检查清单、图集、帧占用和透明单元格，并逐行动画预览 9 种标准动作和 v2 指针注视方向。

独立桌宠

透明无边框、置顶、可拖动、70%–140% 缩放、托盘运行，不依赖 Codex 且不占任务栏。

当前平台成品

一键生成原生运行程序、宠物资源目录和便携 ZIP，适合同平台直接分发。

跨平台构建包

将宠物资源、轻量 builder 和构建指引分发到目标设备，再生成对应平台的原生桌宠。

Docker Web Studio

通过浏览器上传宠物 ZIP，执行同一套格式和安全校验，并下载跨平台构建包。

选择使用方式 ​

  • 直接使用 Studio：从 GitHub Releases 下载当前系统安装包，适合本地导入、预览和打包。
  • 跨平台分发：阅读跨平台构建与分发，选择目标设备 builder 或 GitHub Actions 原生构建。
  • 服务器提供服务：阅读 Docker Web 版，部署浏览器上传和构建包下载入口。
  • 开发和贡献：阅读开发和发布构建，了解本地开发、测试、文档和 Release 产物。
